# Game-Mokepon

Este proyecto es juego de ataque y aleatoridad simple desarrollado con HTML, CSS y JavaScript.

## Tecnologías usadas
- HTML
- CSS
- JavaScript

## Características
- Escoger un personaje, moverlo en un mapa, tener una batalla aleatoria.
- Diseño responsivo.
